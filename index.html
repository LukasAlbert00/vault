Widgetten opdateres automatisk når du logger i appen

1
Host appen på GitHub Pages

Følg installationsguiden og upload filerne til GitHub Pages — du får en URL som ditbrugernavn.github.io/vault

2
iPhone: Brug Scriptable

Download Scriptable gratis fra App Store. Det er en app der lader dig lave egne widgets med JavaScript.

3
Kopier widget-koden herunder

Opret et nyt script i Scriptable og indsæt koden. Skift URL'en til din GitHub Pages-adresse.

4
Tilføj til startskærmen

Hold fingeren nede på startskærmen → + → Scriptable → vælg dit script → færdig!
